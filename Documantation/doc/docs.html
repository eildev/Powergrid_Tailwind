Setup Environment for customization

First you must have Node.js installed in your computer.

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

We are using Tailwind Css which allows having complete automation to build flow. here.

Install dependency

Install all dependency files just run command npm install

When all dependency install then run this command npm run dev. then you can customize this projects

Customization

You can customize various elements including navbar, sidebar, form elements etc by simply editing tailwind css class

Make sure that the npm run dev npm task is running in the terminal to compile input.css files when you make changes. Once the changes are made, the input.css file will be compiled and the changes will be reflected in the browser after an automatic browser reload.

It is important to run npm run dev command from the directory where the node_modules is located.
